How it Decides
The three rules.
01
Round Robin
Cycles servers in order — 1, 2, 3, repeat. Simple and fair.
02
Least Connections
Sends new requests to whichever server is least busy right now.
03
IP Hash
Same user always lands on the same server. Sticky sessions.
Two Flavors
L4 vs L7.
Layer 4
L4
Routes by IP and port. Doesn't read the request.
  • Blazing fast
  • TCP / UDP
  • Protocol-agnostic
Layer 7
L7
Reads HTTP. Routes by URL, header, cookie.
  • Smart routing
  • SSL termination
  • A / B testing
